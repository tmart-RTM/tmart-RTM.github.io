# Release Notes



## 2.4.5 (October 28, 2024)

- Added support for SAFE.zip files 
- Fixed minor path issues


## 2.4.3 (October 26, 2024)

- Added n_jobs parameter to AEC.run for customized multiprocessing 


## 2.3.0 (March 28, 2024)

- Added AEC support for level-1 Landsat 9 OLI-2 products


## 2.2.2 (January 13, 2024)

- Added AEC support for PRISMA L1R files from ACOLITE


## 2.0.0 (October 19, 2023)

- Public release of the adjacency-effect correction (AEC) module for level-1 Sentinel-2 MSI and Landsat 8 OLI products


## 1.3.6 (May 26, 2023)

- Added module for calculating the sea-surface reflectance factor 


## 1.0.0 (March 31, 2022) 

- First public release of T-Mart 
- Support modelling of the adjacency effect
